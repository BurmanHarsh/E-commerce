

# DripMint - Fashion E-Commerce Platform

## Overview
A modern, sleek e-commerce website for streetwear and apparel with a dark theme design, custom product designer, and full shopping experience.

---

## Phase 1: Core Foundation & Design System

### Design Theme
- **Dark background** with subtle gradients
- **Neon accent color** (yellow/green or similar vibrant pop)
- **Clean typography** with bold headings
- **Card-based layouts** for products
- **Smooth animations** and hover effects

### Pages Structure
- Landing/Home page
- Product catalog (browse all)
- Product detail page
- Custom designer page
- Cart & Checkout
- User account pages

---

## Phase 2: Shopify Integration

### Store Setup
- Connect Shopify for product management
- Sync product catalog, inventory, and variants
- Set up categories (Best Sellers, New Arrivals, etc.)
- Handle secure checkout through Shopify

### Product Types
- T-shirts (multiple designs, sizes, colors)
- Hoodies & Sweatshirts
- Pants & Shorts
- Accessories (hats, bags, etc.)

---

## Phase 3: Core Shopping Features

### Product Browsing
- Grid/list view toggle
- Category filtering (Apparel type, Collection)
- Price range filter
- Sort by (Popular, Price, Newest)
- Search functionality

### Product Details
- High-quality image gallery
- Size & color selection
- Quantity picker
- "Add to Cart" functionality
- Related products section

### Shopping Cart
- View/edit cart items
- Update quantities
- Remove items
- Cart summary with totals
- Proceed to Shopify checkout

---

## Phase 4: Custom T-Shirt Designer

### Design Tool Features
- Select base t-shirt (color, style)
- Add text with custom fonts
- Upload custom graphics
- Preview front/back views
- Save design to cart

---

## Phase 5: User Accounts

### Account Features
- Sign up / Login (email-based)
- User profile (name, avatar)
- Order history
- Saved favorites/wishlist
- Shipping addresses

---

## Phase 6: Polish & Extras

### Homepage Elements
- Hero section with featured products
- Promotional banners
- Featured collections
- Newsletter signup

### Mobile Optimization
- Fully responsive design
- Touch-friendly interactions
- Mobile navigation menu

---

## Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS
- **E-commerce**: Shopify (product catalog, checkout, orders)
- **Authentication**: Supabase Auth (user accounts)
- **Database**: Supabase (user profiles, favorites)
